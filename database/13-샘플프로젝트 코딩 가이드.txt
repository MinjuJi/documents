- 고객 관련 메뉴
	1.신규등록
		아이디, 이름, 비밀번호, 이메일을 입력받는다.
		Customer객체를 생성해서 입력값을 담는다.
		서비스의 새 고객 등록 기능을 호출한다.	-> Customer객체 전달
	2.로그인
		아이디, 비밀번호을 입력받는다.
		서비스의 로그인 기능을 호출한다.	-> 아이디, 비밀번호 전달
	3.로그아웃
		서비스의 로그아웃 기능을 호출한다.
	4.주문내역조회
		서비스의 주문내역 조회 기능을 호출해서 주문내역을 조회한다.
		조회된 주문내역을 출력한다.
	5.주문정보 상세보기
		서비스의 주문내역 조회 기능을 호출해서 주문내역을 조회한다.
		조회된 주문내역을 출력한다.
		주문상세정보를 조회할 주문번호를 입력받는다.
		서비스의 주문상세정보 조회 기능을 호출해서 주문상세정보를 조회한다. -> 주문번호 전달
		조회된 주문상세정보를 출력한다.

- 주문 관련 메뉴
	1.주문하기
		서비스의 모든 상품 조회기능을 호출해서 상품정보를 조회한다.
		조회된 상품들을 출력한다.
		
		주문상품들을 저장할 ArrayList<OrderProduct>를 생성한다.
		반복 시작
			
			주문할 상품의 번호와 수량을 입력한다.
			주문할 상품번호로 0을 입력하면 탈출
			OrderProduct객체를 생성해서 상품번호와 수량을 담는다.
			OrderProduct를 위에서 생성한 ArrayList에 저장한다.
		반복 끝
		서비스의 주문하기 기능을 호출한다 -> ArrayList 전달
	2.주문취소
		서비스의 주문내역 조회 기능을 호출해서 주문내역을 조회한다.
		조회된 주문내역을 출력한다.
		취소할 주문번호를 입력받는다.
		서비스의 주문취소 기능을 호출한다. -> 주문번호 전달
	3.주문내역조회
		서비스의 주문내역 조회 기능을 호출해서 주문내역을 조회한다.
		조회된 주문내역을 출력한다.


서비스 
	1. 새 고객 등록 기능
		DAO의 고객아이디로 고객정보 조회 기능을 호출해서 고객정보를 조회한다. -> 아이디 전달
		고객정보가 조회되면 예외를 발생시킨다.
		DAO의 고객등록 기능을 호출한다.	-> Customer 전달
	2. 로그인 기능
		DAO의 고객아이디로 고객정보 조회 기능을 호출해서 고객정보를 조회한다. -> 아이디 전달
		고객정보가 조회되지 않으면 예외를 발생시킨다.
		비밀번호가 일치하지 않으면 예외를 발생시킨다.
		조회된 고객정보를 loginedCustomer에 저장한다.
	3. 로그아웃 기능
		loginedCustomer에 null을 저장해서 저장된 고객정보를 없앤다.
	4. 주문내역 조회 기능
		loginedCustomer가 null이면 예외를 발생시킨다.
		DAO의 주문내역 조회 기능을 호출해서 주문내역을 조회한다. -> 로그인된 사용자의 고객번호 전달
		조회된 주문내역을 반환한다.
	5. 주문상세정보 조회 기능
		loginedCustomer가 null이면 예외를 발생시킨다.
		DAO의 주문정보 상세 조회기능을 호출해서 주문정보를 조회한다. -> 주문번호 전달
		DAO의 주문상품들 조회기능을 호출해서 주문상품들 정보를 조회한다. -> 주문번호 전달
		조회된 주문상품정보를 주문정보에 저장하고 반환한다.
	6. 주문하기 기능
		loginedCustomer가 null이면 예외를 발생시킨다.
		Order 객체를 생성하고, 로그인한 사용자정보를 주문자로 저장한다.
		DAO의 주문번호 조회 기능을 호출해서 신규주문번호를 조회한다.
		조회된 신규주문번호를 Order에 저장한다.
		DAO의 주문정보 저장기능을 호출한다. -> Order객체 전달
		
		총구매가격용 변수
		반복시작
			전달받은 ArrayList<OrderProduct>에서 OrderProduct를 하나씩 꺼낸다.
			OrderProduct에 조회된 신규주문번호를 저장한다.
			DAO의 상품정보 조회기능을 호출해서 상품정보를 조회한다. -> 상품번호
			조회된 상품정보를 OrderProduct에 저장한다.
			상품가격과 할인율로 실제구매가격을 계산한다.
			계산된 구매가격을 OrderProduct에 저장한다.
			DAO의 구매상품 저장기능을 호출한다. -> OrderProduct
		반복종료

		총구매가격으로 적립포인트를 계산한다. (적립률은 0.2%)다
		DAO의 고객정보 조회 기능을 호출해서 고객정보를 조회한다. -> 고객번호 전달
		조회된 고객정보의 포인트에 계산된 적립포인트를 누적시킨다.
		DAO의 고객정보 변경기능을 호출한다.			-> Customer 전달

	7. 주문취소 기능
		loginedCustomer가 null이면 예외를 발생시킨다.
		DAO에서 주문정보 조회기능을 호출해서 주문정보를 조회한다. -> 주문번호 전달	
		주문정보의 상태를 '주문취소'로 변경한다.
		DAO의 주문정보 변경기능을 호출한다. -> Order 전달
