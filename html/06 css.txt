CSS(Cascading Style Sheets)
	- HTML 문서가 스크린, 출력물, 여러 미디어에 어떻게 표현될지를 정의하는 언어다.
	- CSS는 화면의 배치방법, 텍스트의 색상이나 크기, 이미지의의 크기나 위치, 표에 대한 설정 등
          HTML 문서에 대한 디자인적인 요소를 담당한다.
	- CSS의 문법
		h1 {color:blue; font-size:12px;} /* 주석내용 */
		* h1(선택자) - 스타일을 적용하고 싶은 엘리먼트를 지정한다.
		* color:blue(css선언) - 엘리먼트에 적용할 스타일을 선언한다.
                                      - 각 css 선언은 ;로 구분한다.
				      - css선언은 property:value로 구성되어 있다.
		* 주석은 /* 코멘트 */로 작성한다.

	- 작성방법
		* Inline CSS
			- 특정 HTML 태그 하나에만 사용할 CSS를 정의할 때 사용한다.
			- <태그 style="CSS선언; CSS선언">과 같은 방식으로 정의한다.
		* Internal CSS
			- HTML문서에서 이 문서에서만 사용할 CSS를 정의할 때 사용한다.
			- <style>태그 내부에 적용할 스타일을 정의한다.
		* External CSS
			- 스타일을 외부파일에 정의한다.
			- HTML문서에 <link> 태그를 사용해서 css파일을 포함시킨다.
			- 디자인을 변경해야 할 때 css 파일 하나만 변경하면 된다.
		* 유지보수 등을 고려했을 때 가장 좋은 작성방법은 External CSS다.

CSS 선택자
	- HTML문서에서 스타일을 적용할 대상을 선택할 때 사용한다.
	
	- 기본 선택자
	------------------------------------------------------------------------------------------------------------------------------------------------------		
	선택자 			작성예			설명				
	------------------------------------------------------------------------------------------------------------------------------------------------------
	태그 			p {...}			모든 p엘리먼트를 선택
	#아이디			#news {...}		id="news" 인 엘리먼트를 선택
	.클래스			.intro {...}		class="intro"인 모든 엘리먼트를 선택
	*			* {...}			모든 엘리먼트를 선택
	태그, 태그		h1, h2 {...}		모든 h1, 모든 h2엘리먼트를 선택
	태그, .클래스		p, .intro {...}		모든 p엘리먼트, class="intro"인 모든 엘리먼트 선택
	태그.클래스		p.intro {...}		<p class="intro">인 엘리먼트를 선택
	.클래스.클래스		.intro.heading {...}	<p class="intro heading">인 엘리먼트를 선택
	------------------------------------------------------------------------------------------------------------------------------------------------------
	
	- 자손, 자식, 형제 선택자
	------------------------------------------------------------------------------------------------------------------------------------------------------
	태그 태그		div p {...}		<div>
							   <p></p>
							   <p></p>
							<div>
							<div>엘리먼트 안에 있는 <p>엘리먼트가 선택
	.클래스 .클래스		.box .intro {...}	<div class="box">
							   <p class="intro"></p>
							   <p class="intro"></p>
	
							class="box"인 안에 있는 class="intro" 선택
	태그 > 태그		div > p {...}		<div>가 부모엘리먼트인 <p>엘리먼트 선택
	.클래스 > 태그		.box > p {...}		class="box"가 부모인 <p>엘리먼트 선택
	.클래스 > .클래스	.box > .item {...}	class="box"가 부모인 class="item"엘리먼트 선택
	#id > 태그		#section > p {...}	id="section"가 부모인 <p>엘리먼트 선택
	#id > .클래스		#section > .heading{...} id="section"가 부모인 class="heading"엘리먼트 선택
	
	태그 + 태그		h3 + p {...}		<h3>엘리먼트 바로 다음에 있는 <p>엘리먼트 선택
	태그 ~ 태그		h3 ~ p {...}		<h3>엘리먼트 다음에 있는 모든 <p>엘리먼트 선택
	------------------------------------------------------------------------------------------------------------------------------------------------------
	
	- 속성 선택자
	------------------------------------------------------------------------------------------------------------------------------------------------------
	[속성명]		[target]		target속성을 가지고 있는 모든 엘리먼트를 선택
	[속성명=속성값]		[name='username']	name="username"인 모든 엘리먼트를 선택
	[속성명^=속성값]	[href^='http']		href속성의 속성값이 http로 시작하는 모든 엘리먼트 선택
	[속성명$=속성값]	[href$='com']		href속성의 속성값이 com으로 끝나는 모든 엘리먼트 선택
	[속성명*=속성값]	[alt*='사진']		alt속성의 속성값에 '사진'이 포함되어 있는 모든 엘리먼트 선택
	[속성명~=속성값]	[alt~='겨울']		alt속성의 속성값에 '겨울'이라는 단어(공백으로 다른 글자와 구분되는)가 포함되어 있는 모든 엘리먼트 선택	
	------------------------------------------------------------------------------------------------------------------------------------------------------

	- 필터 선택자
	------------------------------------------------------------------------------------------------------------------------------------------------------
	:first-child		p:first-child		<p>엘리먼트중에서 <p>의 부모를 기준으로 첫번째 자식엘리먼트를 선택
	:last-child		p:last-child		<p>엘리먼트중에서 <p>의 부모를 기준으로 마지막번째 자식엘리먼트를 선택
	:nth-child(n)		p:nth-child(2)		<p>엘리먼트중에서 <p>의 부모를 기준으로 두번째 자식 엘리먼트를 선택
	:not(selector)		p:not(:first-child)	<p>엘리먼트중에서 첫번째 자식이 아닌 <p>를 선택
	------------------------------------------------------------------------------------------------------------------------------------------------------
	
	- 가상 선택자
	------------------------------------------------------------------------------------------------------------------------------------------------------
	:hover 			img:hover		<img>엘리먼트중에서 마우스가 올라가 있는 것 선택
	:focus			input:focus		<input>엘리먼트중에서 포커스를 가지고 있는 것 선택
	
	a:link			a:link			<a>엘리먼트 중에서 방문적이 없는 링크
	a:visited		a:visited		<a>엘리먼트 중에서 방문한적인 있는 링크
	a:active		a:active		<a>엘리먼트 중에서 지금 선택한 링크
	-------------------------------------------------------------------------------------------------------------------------------------------------------
