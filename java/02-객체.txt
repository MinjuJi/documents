클래스 작성하기

public class 클래스명 {

	// 속성

	// 생성자

	// 메소드
}

* 클래스명은 식별자다.
* 클래스명은 식별자 작성규칙을 따른다.
* 클래스명은 관례적으로 대문자로 시작한다.

좋은 클래스명
	public class Student { ... }
	public class StudentService { ... }
	public class BankingDepositService { ... }
	public class EmployeeController { ... }

나쁜 클래스명
	public class professor { ... } 		// 대문자로 시작하지 않음
	public class Free_Board { ... } 	// 카멜표기법이 아님
	public class Add { ... }		// 클래스명으로 동사는 적절하지 않음


객체 생성하기
	1. 객체 생성
		new 클래스명();	// new 키워드를 사용해서 객체를 생성한다.
	2. 객체 생성 후 참조변수에 객체의 주소값(참조값) 담아두기
		클래스명 참조변수 = new 클래스명();
		* 객체 생성과정 살펴보기
			- new 클래스명()
				클래스를 메모리에 로딩한다.
				로딩된 클래스로 객체를 생성한다.
				생성된 객체의 생성자를 실행한다.(* 요건 나중에..)
				생성된 객체의 참조값을 반환한다.
			- 클래스명 참조변수
				지정된 클래스의 참조값을 담는 참조변수가 생성된다.
				* 지정된 클래스명 타입의 객체 참조값을 담을 수 있는 참조변수가 생성되는 것임
			- 클래스명 참조변수 = new 클래스명()
				위의 두 과정를 모두 실행하고, 
				참조값이 변수(참조변수)에 대입된다.

3. 객체 사용하기
	클래스명 참조변수 = new 클래스명(); 
	* 객체 생성이 이미 되어 있어야 함.
	
	- 사용하기
		* 생성된 객체의 속성 사용하기
			참조변수.속성 = 값;			
			System.out.println(참조변수.속성);

		* 생성된 객체의 생성자 사용하기	
			객체 생성이 완료된 후에는 사용할 수 없음

		* 생성된 객체의 메소드 사용하기
			참조변수.메소드();
			참조변수.메소드(인자);
