추상클래스
	- 추상메소드를 보유할 수 있는 클래스다.
	- 구현메소드도 보유할 수 있다.
	- new 키워드를 사용해서 객체 생성할 수 없다.

	- 클래스를 정의할 때 abstract 키워드를 추가한다.
	  public abstract class Parent {

	  }
	- 목적
		- 자식클래스들이 반드시 구현할 기능을 추상화한다.
		- 자식클래스들의 구현 부담을 감소시킨다.
		- 템플릿 클래스를 구현할 때 사용된다.

추상메소드
	- 구현부가 없는 메소드다.
          예시
		// 구현부가 없는 메소드 		<--- 추상메소드
		접근제한자 abstract 반환타입 메소드명(매개변수, ...);
		// 구현부는 있지만 수행문이 없는 메소드	<--- 구현메소드
		접근제한자 반환타입 메소드명(매개변수, ...){
		} 
	- 추상클래스, 인터페이스만 보유할 수 있다.
