자바의 주요 API

java.lang
	- 자바 애플리케이션 개발과 관련된 가장 기본적인 라이브러리들을
	  포함하고 있는 패키지
	- 예) String, System, Object, Wrapper 클래스 등
java.util
	- 자료구조관련 프레임워크, 날짜/시간관련 클래스, 국제화, 
          그 외 다양한 유틸클래스를 포함하고 있는 패키지
	- 예) List, Set, Tree, Map, Date, Arrays, Collections, Calendar 
java.io
	- input/output(읽기/쓰기, 입력/출력)과 관련된 라이브러리들을
	  포함하고 있는 패키지
	- 예) InputStream, OutputStream, Reader, Writer, File 
java.net
	- 네트워크 통신과 관련된 라이브러리들을 포함하고 있는 패키지
	- 예) Socket, ServerSocket, URL
java.sql
	- 데이터베이스 엑세스 작업과 관련된 라이브러리를 포함하고 있는 패키지
	- 예) Connection, PreparedStatement, ResultSet, ...
